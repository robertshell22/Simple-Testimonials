INTRODUCTION
------------

This modules creates a testimonial content type, a testimonials list page, a
random testimonial block, and a slider block that rotates through five testimonials.

The display name, page description, and url alias of the testimonials section and block titles
are configurable in a settings page.

Maintainer: Robert Shell (http://www.drupal.org/user/2777399)


INSTALLATION
------------

1. Copy simple_testimonials folder to modules directory.
2. At admin/modules enable the Simple Testimonials module.
3. Enable permissions at admin/people/permissions.
4. Configure the module at admin/config/content/simple_testimonials


CONFIGURATION
--------------

Visit admin/config/content/simple_testimonials to change settings for the
display name, page description, and url alias of the testimonials section and
block titles.
